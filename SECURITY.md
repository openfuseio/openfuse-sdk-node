# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

**Email:** rodrigo@openfuse.io

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Initial assessment:** Within 7 days
- **Resolution target:** Within 30 days (depending on severity)

## Disclosure Policy

- We follow responsible disclosure practices
- We will coordinate with you on public disclosure timing
- Credit will be given to reporters (unless you prefer to remain anonymous)

## Scope

This policy applies to the `@openfuse/sdk` package and this repository.

## Not a Vulnerability?

For general bugs, please use [GitHub Issues](https://github.com/openfuseio/openfuse-sdk-node/issues).
